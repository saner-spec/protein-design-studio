# Protein Design Studio — 最终实施计划 v4

> **核心决策：Phase 1 不整合 ProTherm。用诚实的"ESM打分+透明不确定性"取代虚假的"精确校准"。**
> Phase 2 做私有校准：用户输入自己的实验结果，工具越用越准。

---

## Phase 1（10天冲刺）目标

**用户加载一个PDB → 浏览3D结构 → 点击残基 → 看到ESM对该位置20种氨基酸的打分 → AI解释结果 → 导出突变列表CSV。**

不做的事：
- ❌ ProTherm整合
- ❌ 精确置信区间
- ❌ 多着色方案（只保留链着色和B-factor）
- ❌ PDF导出
- ❌ 移动端适配
- ❌ GFP家族知识综合

## 关键设计决策

### 不确定性声明（替代精确校准）

```
当用户看一个预测时，看到的是：

  🧬 GLU222 → ALA
  
  ESM打分: -0.45 (负值=可能稳定化)
  
  ⚠️ 注意：ESM-2 650M 预测突变效应的整体相关性约
     0.4-0.6（中等）。此分数未经实验数据校准，
     仅供筛选参考，不能替代实验验证。
  
  建议：先做丙氨酸扫描实验验证关键位置，
       再对top候选做详细表征。
```

**不假装精确，诚实说"我不确定"。这本身就是护城河。**

### 批量扫描模式

```
不是逐个点击238个残基。

用户点"扫描表面残基" →
  后台对80个表面暴露位批量ESM打分 →
  前端显示热图: 每行=一个残基, 每列=一种aa, 颜色=分数
  →
  用户点热图中感兴趣的格子 → 跳到3D视图 → 
  AI开始解释
```

### 私有校准（Phase 2，10天后）

```
用户测了10个突变，回来输入实验结果：

  E222A: 测出来 ΔΔG = -0.3 (ESM预测 -0.45)
  E222D: 测出来 ΔΔG = -1.1 (ESM预测 -0.80)
  ...

工具 → 对剩余位置重新打分，偏差缩小 →
  "已用10个实验点校准，对本蛋白预测准确度提升约30%"

这才是真正的壁垒——越用越准，别人复制不了。
```
